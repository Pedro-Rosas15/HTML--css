Introdução

Bem-vindo ao curso de HTML & CSS. Este capítulo apresenta os conceitos fundamentais: o que é HTML, por que usamos semântica, e como o CSS separa estrutura de apresentação.

Por que aprender HTML?

HTML é a linguagem de marcação que estrutura o conteúdo na web. Entender sua semântica melhora a acessibilidade, a indexação pelos mecanismos de busca e a manutenção do código.

Exemplo — Estrutura básica
// código abaixo
<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="utf-8">
    <title>Meu site</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
  </body>
</html>

Significado: indica a estrutura básica de um documento HTML5 — o que todo arquivo deve ter no começo.
